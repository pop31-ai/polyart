# Changelog

## v2.1.0 (2026-07-12)
- Added polyart_biology.py: Growth curves, phyllotaxis, biomechanics, Turing patterns, variant generator
- Added polyart_sculpture.py: LatheBody, MuscleRelief, PiranesiArch, RomanSymbols, GameAssets
- 10 showcase examples in examples/ directory
- 5 technical articles in articles/ directory

## v2.0.0 (2026-07-12)
- Complete API rewrite with fluent/chainable pattern
- GreekLines, RomanLines, GoldenRatio, PolyCoeffs, SuperFormula classes
- Templates: Apollo, Greek vase, Parthenon, golden composition, etc.
- Full Doxygen documentation

## v1.0.0 (2026-07-12)
- Initial polyart_format.py: save/load/render .polyart files
- polyart_editor.py: tkinter visual editor
- polyart_viewer.py: lightweight viewer
- Basic API with Canvas, PolyObj, Layer classes
- .polyart JSON format specification
